# acm-hash

takes string and returns has as a 32 bit hex-code 

## Usage

## Installation

```sh
  $ git clone https://www.github.com/incessantemeraki/acm-hash.git
```

## Example

```js
  var acm-hash = require('./acm-hash')
  var str = 'hello world'
  console.log(acm-hash(str)) // 161b1212
```
